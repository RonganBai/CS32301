    Career Introduction

    "Arknights" has been upgraded and improved on the basis of the game mode of Plants vs. Zombies. The difference is that Tomorrow's Ark has 8 occupations, and there are many branches of the same occupation. It's as if computer science includes data engineering and game design. Unlike Plants vs. Zombies, a single plant can be used multiple times. They are unique among all the characters in Arknights. Character upgrades can only be done before the game starts.
